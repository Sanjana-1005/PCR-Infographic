# PCR-Infographic – Graphical Abstract

 Title
PCR Amplifies DNA via Cyclic Temperature-Controlled Reactions

📄 Research Paper Used

Polymerase Chain Reaction (PCR): A Short Review
Authors: MT Rahman et al.
Published in: AKMMC Journal (2013)

Objective

To visually represent the process of Polymerase Chain Reaction (PCR) in a simplified and easy-to-understand graphical format for academic purposes.

Description of the Infographic

This infographic illustrates the PCR mechanism in three main stages:

1. Sample Input

- DNA sample is extracted and prepared for amplification.

2. PCR Cycle (25–35 cycles)

The process includes three repeated steps:

- Denaturation (94–96°C): DNA strands separate
- Annealing (45–60°C): Primers bind to target sequences
- Extension (72°C): Taq polymerase synthesizes new DNA

These steps repeat multiple times, leading to exponential amplification.

3. Output

- Millions to billions of DNA copies are generated.


 Applications of PCR

- Disease diagnosis (HIV, hepatitis, infections)
- Cancer detection
- Genetic fingerprinting and paternity testing
- Research applications (gene cloning, mutation analysis)


Tool Used

- BioRender (for creating infographic)

📚 Conclusion

PCR is a powerful molecular biology technique that enables rapid and exponential amplification of DNA, making it essential in diagnostics, research, and biotechnology.
